The Telco Customer Churn dataset can be downloaded from kaggle with url https://www.kaggle.com/blastchar/telco-customer-churn.
The dataset we chosen is a compliation of datas and metadatas of the subscription status of the telecommunication industry, with 7043 rows (customers) and 21 columns (features), with "Churn" as our target response of the process.
